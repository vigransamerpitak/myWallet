📖 คู่มือการใช้งานระบบ "คู่รักนักออม" 👩‍❤️‍👨
(สำหรับส่งให้แฟนอ่านเข้าใจใน 2 นาที)

ยินดีต้อนรับสู่ระบบกระเป๋าเงินจำลองส่วนตัวของเราสองคนนะ! เว็บนี้ทำขึ้นมาเพื่อให้เราสามารถช่วยกันบริหารเงิน ออมเงินฉุกเฉิน และเคลียร์บิลส่วนกลางกันได้ง่ายๆ โดยแบ่งออกเป็น 4 กระเป๋าหลักๆ ดังนี้จ้า:

💰 1. รู้จักกับ 4 กระเป๋าเงินของเรา
กระเป๋าของฉัน 🙋‍♂️ : เงินส่วนตัวของฝั่งผู้ชาย (ไม่เกี่ยวกับเงินกองกลาง)

กระเป๋าของแฟน 🙋‍♀️ : เงินส่วนตัวของฝั่งผู้หญิง (ไม่เกี่ยวกับเงินกองกลาง)

เงินกองกลางร่วมกัน 🤝 : กระเป๋าสำหรับใช้จ่ายเรื่องของเราสองคน (เช่น ค่าอาหารที่กินด้วยกัน, ค่าน้ำมันทริปเที่ยว, ค่าของสดเข้าตู้เย็น)

เงินออมสำรองฉุกเฉิน 🎯 : คลังเงินเซฟโซนห้ามแตะ! เก็บไว้ใช้ยามจำเป็นจริงๆ เช่น บาดเจ็บ เกิดอุบัติเหตุ หรือต้องการรักษาพยาบาลด่วน

✍️ 2. วิธีบันทึกรายการเงินใหม่ (จดไวใน 5 วินาที)
ระบบนี้ใช้ระบบ Fast Click (ไม่มีปุ่มบันทึกแยก) พิมพ์ตัวเลขเสร็จแล้วจิ้มหมวดหมู่จะบันทึกทันทีจ้า!

ใส่จำนวนเงิน ในช่องแรก (เช่น 500)

ใส่บันทึกช่วยจำ ในช่องกลาง (เช่น ค่าชาบูมื้อเย็น)

เลือกกระเป๋าเงิน ในช่องขวาสุด (สำคัญมาก!):

ถ้ากินด้วยกัน ให้เลือก 🤝 ใช้ร่วมกัน (หารครึ่ง/กองกลาง)

ถ้าจ่ายเงินส่วนตัว ให้เลือก 🙋‍♂️ ของฉันคนเดียว หรือ 🙋‍♀️ ของแฟนคนเดียว

ขั้นตอนสุดท้าย: กดจิ้มที่ปุ่มหมวดหมู่สีชมพู 🔴 (รายจ่าย) หรือสีเขียว 🟢 (รายรับ) ได้เลย ยอดเงินจะถูกบันทึกและคำนวณเข้าตารางทันที!

🎯 3. ระบบ Checklist ภารกิจการเงินประจำเดือน
ในแต่ละเดือนเราจะมีเควสจำลองช่วยกันออมเงินและจ่ายบิลประจำ:

ภารกิจ [ออม] 🎯 : ถ้าเราเก็บเงินก้อนนั้นได้แล้ว ให้มากดติ๊กถูก ✅ หน้าช่อง ระบบจะทำการ โยกเงินก้อนนั้นเข้าสู่กล่องเงินออมสำรองฉุกเฉิน 🎯 ให้เราโดยอัตโนมัติ ทันที!

ภารกิจ [บิล] 📄 : ถ้าจ่ายค่าส่วนกลางหรือค่าน้ำค่าไฟแล้ว ให้มากดติ๊กถูก ✅ ระบบจะทำการ หักเงินออกจากกระเป๋าส่วนกลาง 🤝 อัตโนมัติ ไม่ต้องไปนั่งพิมพ์จดซ้ำ

🤝 4. สิ้นเดือนเคลียร์ตังค์กันยังไง?
ให้สังเกตที่กล่องสีม่วงด้านบน "สรุปบิลหารครึ่งประจำเดือน" * ทุกๆ รายการที่เราเลือกผูกกับกระเป๋า 🤝 ใช้ร่วมกัน ระบบจะเอามาบวกรวมกันทั้งหมดในเดือนนั้นๆ แล้วหารครึ่งให้อัตโนมัติ

สิ้นเดือนมาเปิดดูกล่องนี้ จะรู้ทันทีว่าเดือนนี้เราสองคนควักเนื้อจ่ายส่วนกลางไปเท่าไหร่ และต้องเอาเงินมาเติมคนละกี่บาทเพื่อให้บัญชีกลับมาสมดุลและเจ๊ากันแบบแฮปปี้ทั้งคู่จ้า!

✏️ 5. พิมพ์ผิด พิมพ์เบิ้ล ทำยังไง?
ไม่ต้องตกใจ! เลื่อนลงมาที่ตารางประวัติด้านล่างสุด:

ถ้าพิมพ์ตัวเลขผิด: ให้กดปุ่ม ✏️ แก้ ฟอร์มด้านบนจะเปลี่ยนเป็นสีเหลืองออโต้ ให้แก้ไขตัวเลขแล้วกด 💾 บันทึกการแก้ไข

ถ้าต้องการลบทิ้ง: กดปุ่ม 🗑️ ลบ แล้วกดยืนยัน รายการนั้นจะหายไปจากระบบทันทีล่ะ

คราวนี้ส่งคู่มือนี้ให้แฟนปั๊บ แฟนเข้าใจวิธีเล่นระบบล็อกอินและสับเปลี่ยนฟิลเตอร์แน่นอนครับ ยิงปืนนัดเดียวได้นกสองตัว ทั้งโชว์สกิล Dev และสร้างวินัยการเงินคู่รักไปด้วยกันเลยครับ! 👩‍❤️‍👨🔥
